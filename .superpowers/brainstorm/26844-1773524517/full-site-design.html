2. Services Page (/services)

Layout: Full-width header + expanded 2x2 card grid
Each card includes: Icon, title, detailed description, list of technologies used, example use cases
Services:
  • AI & Machine Learning — Custom models, NLP, recommendation engines, predictive analytics, LLM integration
  • Data Engineering — ETL/ELT pipelines, data warehousing, real-time streaming, orchestration (dbt, Airbyte, Prefect)
  • Analytics & Insights — Dashboards, BI tools, data modeling, custom reporting, KPI tracking
  • Full Stack Development — Web applications, REST/GraphQL APIs, cloud infrastructure, CI/CD, end-to-end product builds

3. About Page (/about)

Sections:
  • Company Story — Mission statement, what drives SWA, approach to data
  • Team — Team member cards with photo placeholders, roles, brief bios
  • Values/Approach — 3-4 pillars (e.g., "Data-Driven", "Production-Grade", "AI-First", "Client-Focused")

4. Careers Page (/careers)

Sections:
  • Why Join Us — Company culture, tech stack, growth opportunity
  • Open Positions — Card-based job listings (title, type, brief description)
  • Application CTA — Email link for applications
Note: Positions can be hardcoded initially and updated via code as needed.

5. Contact Page (/contact)

Layout: Clean centered layout with:
  • Headline & tagline — "Let's Work Together"
  • Email CTA button — Large, prominent mailto: link button
  • Social links — GitHub, LinkedIn icons

Technical Architecture

  • Router: TanStack Router with file-based routing (SPA mode, not SSR)
  • UI: shadcn/ui components (base-nova style, dark mode)
  • Styling: Tailwind CSS v4 with OKLch color tokens, dark theme default
  • Build: Vite + Bun, base: '/' for custom domain
  • Deploy: GitHub Pages via GitHub Actions, 404.html fallback for SPA routing
  • Animations: CSS-only transitions, scroll-triggered reveals via Intersection Observer
  • Responsive: Mobile-first, hamburger nav on small screens
